# Implementing a Celo-Based Decentralized Domain Name System

- contracts/: This folder contains the Solidity smart contract files.

- DomainRegistry.sol: The main smart contract for domain registration, transfer, and dispute resolution.

- migrations/: This folder contains the migration files for deploying the smart contracts.

1_initial_migration.js: Truffle migration script for deploying the initial migration contract.
2_deploy_contracts.js: Truffle migration script for deploying the DomainRegistry contract.

- test/: This folder contains the test files for the smart contracts.

- DomainRegistry.test.js: Test file for the DomainRegistry contract.

- truffle-config.js: Truffle configuration file specifying the network settings for deployment.

- package.json: The npm package file containing the project dependencies and scripts.
